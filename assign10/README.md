Minimum Spanning Tree
----------------------

1. Find a Minimum Spanning Tree using Kruskal's algorithm.
2. Find a Minimum Spanning Tree using Prim's algorithm.

Sample Inputs
-------------------
>Sample inputs for problems 1 and 2.

**Sample Input**
```
7
0 0 7 5 0 0 0
0 0 8 0 5 0 0
7 8 0 9 7 0 0
5 0 9 0 15 6 0
0 5 7 15 0 8 9
0 0 0 6 8 0 11
0 0 0 0 9 11 0
```
**Sample Output**
```
0 0 7 5 0 0 0
0 0 0 0 5 0 0
7 0 0 0 7 0 0
5 0 0 0 0 6 0
0 5 7 0 0 0 9
0 0 0 6 0 0 0
0 0 0 0 9 0 0
```
